KeepScore - Tine Minte Scorul
Aplicatie usor de utilizat pentru Android

Trebuie sa tii scorul? Arunca pixul si hartia!

KeepScore este o aplicatie pentru scor gratuita si open-source care poate fi folosita pentru orice joc bazat pe puncte.

Features:

- Salveaza scorul automat

- Arata istoricul scorului pentru fiecare joc

- Istoricul se salveaza automat dupa 10 secunde (configurabil)

- Mentine ecranul deschis

- Permite adaugarea a 2-8 jucatori

- Salveaza jocuri multiple

- Long-press on a button to add/subtract different values

- Configure the add/subtract buttons

- Adauga/editeaza/sterge jucatori in timpul jocului

- Backup/restore pe cardul SD

Enjoy!
